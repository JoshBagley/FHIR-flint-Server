Industry CDC Census 2012	PHVS_Industry_CDC_Census2012	2.16.840.1.114222.4.11.8027	1	2012 Industry coding system used by CDC (NIOSH & NCHS) for coding industry text. Industry describes an economic/business sector comprised of businesses/ enterprises concerned with the output of a specified category of products or services (e.g., the construction industry or the agriculture industry). This industry code system includes 2012 U.S. Census Bureau industry codes and additional codes developed by CDC for military and unpaid workers. The 2012 Census industry categories are based on the 2012 North American Industry Classification System (NAICS). The PH_Occupation_CDC_Census2012 code system should be used in conjunction with this industry code system when coding both industry and occupation.
