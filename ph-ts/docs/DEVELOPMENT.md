# Development Guide

## Prerequisites

- Docker 20.10+
- Docker Compose 2.0+
- Python 3.11+
- Node.js 18+
- Git

## Setup

1. Clone repository
2. Run setup script: `./scripts/setup.sh`
3. Start services: `make start`
4. Access application
